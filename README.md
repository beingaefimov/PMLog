# PMLog

Факт о выполнении, параметры выполнения плановые и фактические, актуализированное плановое значение, сопутствующая информация, решения в чатах и т.д. - всё это идёт непрерывным "логом" (сплошным текстом или структурированно). Этот пакет использует LLM для анализа этих "логов", выяснения проблем, формирования позиции для вероятных корректировок.

The fact of the execution, the planned and actual execution parameters, the updated planned value, related information, chat decisions, etc. - all this goes in a continuous "log" (plain text and structured). This package uses LLM to analyze these "logs", identify problems, and form a position for possible adjustments.
